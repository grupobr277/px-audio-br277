# 🎤 PX Audio System - BR-277

Sistema de comunicação por áudio em tempo real para motoristas da BR-277.

## 🚀 Funcionalidades

- ✅ 5 estações de rádio (Curitiba, Campo Largo, Fazenda Rio Grande, Morretes, Paranaguá)
- ✅ Gravação de áudio de até 20 segundos
- ✅ Contagem regressiva visual
- ✅ Notificações em tempo real
- ✅ Histórico das últimas mensagens
- ✅ Interface bonita e responsiva

## 🛠️ Tecnologias

- Node.js + Express
- Socket.io (WebSockets)
- HTML5 + CSS3 + JavaScript

## 📦 Como executar

1. Clone o repositório
2. Execute `npm install`
3. Execute `npm start`
4. Acesse `http://localhost:3001`

## 📡 Estações disponíveis

- Curitiba - Início da BR-277
- Campo Largo - Km 30
- Fazenda Rio Grande - Km 15
- Morretes - Km 70
- Paranaguá - Porto
